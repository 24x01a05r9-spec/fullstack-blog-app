# fullstack-blog-app
Blog Platform with Comments

Overview

Blog Platform with Comments is a full-stack web application that allows users to create, edit, and manage blog posts while interacting through a comment system. The platform includes user authentication, RESTful APIs, and database integration to provide a complete blogging experience.

Features

* User Registration and Login
* Secure Authentication System
* Create Blog Posts
* Edit Existing Posts
* Delete Posts
* View All Blog Posts
* Comment on Blog Posts
* RESTful API Integration
* Database Storage and Management

Technologies Used

Frontend

* HTML
* CSS
* JavaScript
* Bootstrap

Backend

* Python (Flask)

Database

* SQLite / MySQL

Installation

1. Clone the repository

git clone https://github.com/yourusername/blog-platform-with-comments.git

2. Navigate to the project directory

cd blog-platform-with-comments

3. Install dependencies

pip install -r requirements.txt

4. Run the application

python app.py

5. Open the application in your browser.

API Endpoints

Authentication

* POST /register
* POST /login

Blog Posts

* GET /posts
* POST /posts
* PUT /posts/
* DELETE /posts/

Comments

* GET /comments/<post_id>
* POST /comments
